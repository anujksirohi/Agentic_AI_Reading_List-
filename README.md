## 🧭 Overview

This repository is a consolidated reference for developers, researchers, and students working with **agentic AI systems**. All five course modules are included as structured PDF files, organized for easy revision and quick lookup.

Whether you're building your first tool-using agent or designing production-grade multi-agent architectures, these notes are meant to be a clean, accessible companion you can return to anytime.

## 📚 Module Details

### 🟦 Day 1 — Introduction to Agents
> `Introduction to Agents.pdf`

- Evolution from predictive AI → autonomous agentic systems
- Core agent architecture (Model · Tools · Orchestration)
- The 5-step agentic problem-solving loop
- Taxonomy of agents (Level 0 → Level 4)
- AgentOps fundamentals: logging, debugging, deployment
- Multi-agent system design patterns

---

### 🟩 Day 2 — Agent Tools & Interoperability with MCP
> `Agent Tools & Interoperability with Model Context Protocol (MCP).pdf`

- Understanding "tools" in agent design
- Built-in tools, function tools, and agent tools
- MCP: the Model Context Protocol for tool interoperability
- Designing safe tool schemas
- Avoiding tool shadowing & misuse
- Secure, scalable tool integrations across ecosystems

---

### 🟨 Day 3 — Context Engineering: Sessions & Memory
> `Context Engineering_ Sessions & Memory.pdf`

- Full breakdown of context engineering
- Understanding sessions (conversation history + working memory)
- Types of memory: short-term, long-term, multimodal, procedural
- Memory extraction, consolidation, and retrieval
- Multi-agent session management patterns
- Production considerations for memory systems

---

### 🟧 Day 4 — Agent Quality
> `Agent Quality.pdf`

- Why evaluating agents ≠ evaluating LLMs
- The four pillars of agent quality:
  - **Effectiveness**
  - **Efficiency**
  - **Robustness**
  - **Safety & Alignment**
- LLM-as-Judge evaluation
- Observability: logs, metrics, traces
- The Quality Flywheel for continuous improvement
- Guardrails & prompt-injection resistance

---

### 🟥 Day 5 — Prototype to Production
> `Prototype to Production.pdf`

- Evaluation-gated CI/CD pipelines
- Pre-production checks, golden datasets, automated testing
- Safe rollout strategies (Canary · Blue-Green · A/B · Feature Flags)
- AgentOps lifecycle: **Observe → Act → Evolve**
- Security foundations & SAIF
- Multi-agent orchestration & the A2A protocol
- Scaling agents with cloud infrastructure
- Operating cost, latency & tool orchestration in production

## 🛠️ Recommended Study Flow

```text
Day 1 → Day 2 → Day 3 → Day 4 → Day 5
```

1. **Start with Day 1** to understand what agents truly are.
2. **Move to Tools & MCP** to learn how agents act on the world.
3. **Study Context Engineering** to master sessions, memory & context windows.
4. **Learn Agent Quality** to evaluate behavior & detect failures.
5. **Finish with Prototype → Production** to understand operational deployment.
